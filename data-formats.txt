Why Do We Need Data Formats?
A data format refers to the structure and arrangement in which
data is organized, stored and presented. Data formats define how
data elements are represented to ensure it is accurately
exchanged, processed & interpreted by computer systems
& applications.

What Is JSON?

2) - JSON is a Data Format
3) - JSON is a text based format
4) - JSON is used to exchange or store information
